HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The charset sets the character encoding.
  • The http-equiv sets an http header for content.
  • The viewport sets the default view for our webpage.
  • The meta contains information about the page that is used by the browser.
  • The element refers to the entire element from opening tag to closing tag.
  • The tag refers only to what appears inside the angle brackets.
  • The id and class is a way to identify elements and select them.

Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git checkout: switches to a different branch.
  • git add -A: command to commit all files in the working branch.
  • git commit -m: short descriptive messages of the changes made.
  • git rm -- cached file name: removes a file from the github repository but keeps it on the local computer.
